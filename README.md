> # **Online Book Store**

## Overview
The Online Book Store project analyzes book sales, customer behavior, and order data to identify purchasing patterns and business insights. The goal of this project is to transform raw transactional data into meaningful insights using data analysis techniques.

## Dataset
The dataset consists of three CSV files:
- Books – Contains information about books such as title, author, category, and price.
- Customers – Includes customer details like name and location.
- Orders – Contains order transactions including customer ID, book ID, quantity, and purchase details.

## Tools & Technologies
- Excel
- SQL

## Project Workflow
1. Data collection and preparation using Excel.
2. Data cleaning and preprocessing.
3. Exploratory Data Analysis (EDA).
4. Data analysis using SQL queries (joins, aggregations, filtering).

## Key Insights
- Identified best-selling books and high-demand categories.
- Analyzed customer purchasing behavior and order frequency.
- Observed sales trends and revenue patterns across orders.
